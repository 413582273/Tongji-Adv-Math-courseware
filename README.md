# Tongji-Adv-Math-Courseware

## 《高等数学》第七版辅助资料

### 简介
本项目旨在提供对同济大学《高等数学》第七版内容的深入理解和辅助。主要内容包括各章节的重要计算过程展示，用于课堂演示，旨在帮助学生更好地掌握和理解高等数学的核心概念和计算方法。

### 内容结构
本项目的内容安排按照《统计大学高等数学第七版》的章节顺序进行。每个文件或文件夹的编号“A-B”对应教材中的第“A”章第“B”小节。

例如：
- `1-2`：代表第1章第2小节的相关资料。
